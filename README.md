## Overview

**Sponsorship Rules** is a reference repository maintained by **AI Forensic Agents**. It exists to solve a single problem: sponsorship ad assets that get rejected or underperform because they fail to meet platform specifications.

This guide covers the **three deliverables** every sponsor must provide:

| # | Asset | Key Constraint |
|---|-------|---------------|
| 1 | **Static Image** | 1800 × 1050 px · PNG/JPG · ≤ 400 KB |
| 2 | **Text Paragraph** | ≤ 360 characters (including one hyperlink CTA) |
| 3 | **Landing Page** | Single destination URL shared across all clickable elements |

Follow the specifications below, review the examples, run through the checklist, and submit with confidence.

## Quick Start

```text
1. Read the three specification sections below.
2. Create your static image at 1800 × 1050 px (PNG or JPG, ≤ 400 KB).
```

## Asset Specifications

### Static Image Requirements

The static image is the visual centerpiece of the sponsorship placement. It must be eye-catching, on-brand, and precisely sized.

| Property | Specification |
|----------|--------------|
| **Dimensions** | 1800 × 1050 px |
| **Aspect Ratio** | ~1.714 : 1 (≈ 12 : 7) |
| **File Format** | PNG or JPG |
| **Maximum File Size** | 400 KB |

#### Detailed Notes

- **Dimensions are exact.** Do not submit images that are "close" (e.g., 1800 × 1048). Platforms may reject assets or stretch/crop them unpredictably.
- **PNG** is recommended for graphics, text overlays, logos, and illustrations where sharp edges matter.
- **JPG** is recommended for photographic content or complex gradients where file size is a concern.
- **400 KB is a hard ceiling.** If your PNG exceeds this, consider:
  - Reducing the number of colors.
  - Converting to JPG at 85–90% quality.
  - Using an image compression tool (see [Tools & Resources](#tools--resources)).

> **⚠️ Important:** Do not add excessive text to the image. Many platforms follow a guideline similar to the "20% text rule." Keep text minimal — let the paragraph copy do the heavy lifting.

```
┌─────────────────────────────────────┐
│                                     │
│          1800 px (width)            │
│                                     │
│   ┌─────────────────────────────┐   │
│   │                             │   │
│   │    Your Visual Content      │   │  1050 px
│   │    Logo · Headline · CTA    │   │  (height)
│   │                             │   │
│   └─────────────────────────────┘   │
│                                     │
│   Format: PNG or JPG  ≤ 400 KB     │
└─────────────────────────────────────┘
```

### Text Paragraph Requirements

The text paragraph accompanies the image and provides context, value proposition, and a call-to-action.

| Property | Specification |
|----------|--------------|
| **Maximum Length** | 360 characters (total, including the hyperlink) |
| **Hyperlinks** | Exactly one (1) — your CTA |
| **CTA Placement** | Ideally toward the end of the paragraph |
| **Emojis** | ❌ Not allowed |
| **Bullet Points** | ❌ Not allowed |
| **Point of View** | Third person preferred (avoid "we," "us," "our") |

#### Character Counting Rules

The 360-character limit includes **everything**:
- All visible text characters
- Spaces
- Punctuation
- The anchor text of the hyperlink

> **Note:** The underlying URL itself (the `href`) typically does **not** count toward the character limit — only the visible/anchor text does. However, always confirm with the specific platform's guidelines.

#### Writing in Third Person

| ❌ Avoid (First Person) | ✅ Preferred (Third Person) |
|------------------------|---------------------------|
| "We built a tool that…" | "AI Forensic Agents built a tool that…" |
| "Our platform helps…" | "The platform helps…" |
| "Try us free for 30 days." | "Try AI Forensic Agents free for 30 days." |
| "Download our guide." | "Download the guide." |

### Landing Page Requirements

Every clickable element in the sponsorship unit must resolve to the **same destination URL**.

| Clickable Element | Destination |
|-------------------|-------------|
| Sponsor name in the ad introduction | → Your Landing Page URL |
| The static image | → Your Landing Page URL |
| The hyperlinked CTA in the text paragraph | → Your Landing Page URL |

> **All three elements point to the same URL.** Do not use different URLs for the image and the text CTA.

#### What Makes a High-Performing Landing Page

Performance tends to improve when the landing page delivers **immediate, tangible value** to the reader. Consider the following content strategies:

| Strategy | Example |
|----------|---------|
| **Educational Content** | A guide, whitepaper, or in-depth article |
| **Free Tool / Trial** | Limited-time access to a product or feature |
| **Discount / Coupon** | An exclusive offer for the newsletter audience |
| **Interactive Resource** | A quiz, calculator, or assessment |
| **Template / Download** | A ready-to-use template relevant to the audience |

> **💡 Pro Tip:** Landing pages that feel like valuable content rather than hard sales pitches tend to achieve higher click-through rates and lower bounce rates. Lead with value — the conversion will follow.

## Best Practices & Tips

### Image Design

1. **Use high contrast.** The image may appear in both light and dark email clients. Test legibility in both.
2. **Keep text on the image minimal.** A short headline or tagline is fine; a full paragraph is not.
3. **Include your logo.** Brand recognition starts here.
4. **Use a clear visual hierarchy.** The reader should understand the message within 2–3 seconds.
5. **Design for mobile.** Over 60% of newsletter opens happen on mobile devices. Ensure text on the image is large enough to read on small screens.

### Copywriting

1. **Front-load the value.** The first sentence should immediately communicate what the reader gains.
2. **Write for scanners.** Assume the reader will spend fewer than 5 seconds on your paragraph.
3. **Use concrete numbers.** "Used by 12,000 security teams" is stronger than "Used by many teams."
4. **End with a clear CTA.** The hyperlink should use action-oriented anchor text (e.g., "Get the free guide" rather than "Click here").
5. **Stay under 340 characters.** Aim for 340 to give yourself a 20-character buffer for last-minute edits.

### Landing Page

1. **Match the message.** The landing page headline should directly reflect the ad copy.
2. **Minimize friction.** Reduce form fields, avoid pop-ups, and make the value accessible quickly.
3. **Ensure fast load times.** A slow page kills conversion. Target under 3 seconds.
4. **Mobile-optimize.** The same mobile-first principle applies here.

## Common Mistakes to Avoid

> Avoid these frequently seen errors that lead to asset rejection or poor performance.

| # | Mistake | Why It's a Problem |
|---|---------|-------------------|
| 1 | **Image is 1800 × 1080 px** (instead of 1050) | Wrong dimensions; will be rejected or cropped |
| 2 | **File size exceeds 400 KB** | Hard limit; asset will not be accepted |
| 3 | **Using GIF, SVG, or WebP format** | Only PNG and JPG are accepted |
| 4 | **Text paragraph exceeds 360 characters** | Copy will be truncated or rejected |
| 5 | **Including emojis in the text** | Emojis are explicitly prohibited |
| 6 | **Using bullet points or line breaks** | The paragraph must be a single, continuous block of text |
| 7 | **Writing in first person ("we," "our")** | Third person is the required voice |
| 8 | **Multiple hyperlinks in the paragraph** | Only one hyperlink (your CTA) is allowed |
| 9 | **CTA at the beginning of the paragraph** | CTA should be placed toward the end for optimal flow |
| 10 | **Different URLs for image and text CTA** | All clickable elements must link to the same destination |
| 11 | **Landing page with no clear value proposition** | Leads to high bounce rates and poor campaign performance |
| 12 | **Submitting a homepage as the landing page** | A dedicated, campaign-specific page almost always outperforms |

## Pre-Submission Checklist

Use this checklist before submitting your sponsorship assets. Every item must pass.

### Static Image

- [ ] Dimensions are **exactly** 1800 × 1050 px
- [ ] File format is **PNG** or **JPG**
- [ ] File size is **≤ 400 KB**
- [ ] Image is visually clear and legible on mobile
- [ ] Logo / brand name is visible
- [ ] Text on image is minimal and readable
- [ ] No transparency issues (if JPG, ensure no alpha channel artifacts)

### Text Paragraph

- [ ] Total character count is **≤ 360** (including hyperlink anchor text)
- [ ] Contains **exactly one** hyperlink
- [ ] CTA hyperlink is placed **toward the end** of the paragraph
- [ ] Written in **third person** (no "we," "us," "our")
- [ ] Contains **no emojis**
- [ ] Contains **no bullet points** or list formatting
- [ ] Copy is a **single paragraph** (no line breaks)
- [ ] Anchor text is action-oriented and clear
- [ ] Grammar and spelling have been proofread

### Landing Page

- [ ] URL is **functional** and loads correctly
- [ ] Page loads in **under 3 seconds**
- [ ] Page is **mobile-optimized**
- [ ] Content delivers **clear value** (guide, tool, coupon, article, etc.)
- [ ] Headline on the page **matches** the ad messaging
- [ ] **Same URL** is used for the sponsor name, image, and CTA hyperlink

### Final Verification

- [ ] All three assets have been reviewed by at least one other team member
- [ ] Assets comply with the platform's advertising policies and terms of service
- [ ] Backup copies of all assets have been saved
