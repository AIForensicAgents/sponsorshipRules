# Sponsorship Asset Submission Checklist

Use this checklist before submitting your sponsorship assets.

## Static Image
- [ ] Dimensions are exactly 1800 x 1050 px
- [ ] File format is PNG or JPG
- [ ] File size is under 400 KB
- [ ] Image is clear and high quality
- [ ] Brand logo/name is visible
- [ ] No misleading or clickbait imagery

## Text Paragraph
- [ ] Total character count is 360 or fewer (including hyperlink)
- [ ] Contains exactly one hyperlink (CTA)
- [ ] CTA is placed toward the end of the paragraph
- [ ] No emojis used
- [ ] No bullet points used
- [ ] Written in 3rd person (no "we/us/our")
- [ ] Grammar and spelling checked
- [ ] Message is clear and compelling

## Landing Page
- [ ] Destination URL is functional and loads correctly
- [ ] Sponsor name in ad intro links to destination URL
- [ ] Static image links to destination URL
- [ ] CTA hyperlink links to destination URL
- [ ] All three links point to the SAME destination URL
- [ ] Landing page provides value (guide, article, coupon, free tool, etc.)
- [ ] Page is mobile-responsive
- [ ] Load time is reasonable (under 3 seconds)

## Final Review
- [ ] All assets are consistent in branding and messaging
- [ ] Assets comply with platform advertising policies
- [ ] A second person has reviewed all materials
